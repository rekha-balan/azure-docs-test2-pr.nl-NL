Create and manage a cloud-based high-performance computing (HPC) cluster by taking advantage of [Microsoft HPC Pack](https://technet.microsoft.com/library/jj899572.aspx) and Azure compute and infrastructure services. HPC Pack is Microsoft's free HPC solution built on Microsoft Azure and Windows Server technologies and supports a wide range of HPC workloads.

For more options to run large-scale parallel, batch, and HPC workloads in Azure, see [Technical resources for batch and high-performance computing](../articles/batch/big-compute-resources.md).
