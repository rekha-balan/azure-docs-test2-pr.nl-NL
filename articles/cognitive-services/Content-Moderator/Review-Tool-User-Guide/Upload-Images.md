# <a name="upload-images"></a>Upload Images #

To explore the user interface, upload up to five images at a time, each image not to exceed 2 MB in file size. Be sure to check your file sizes before uploading.

![Upload Images](https://docstestmedia1.blob.core.windows.net/azure-media/articles/cognitive-services/Content-Moderator/Review-Tool-User-Guide/images/1-Try-Image-1.PNG)

## <a name="submit-for-moderation"></a>Submit for Moderation ##
If you end up uploading more than five images at a time, use the Trashcan icon to remove unwanted images. Then, submit the images for automated moderation.

![Submit Images](https://docstestmedia1.blob.core.windows.net/azure-media/articles/cognitive-services/Content-Moderator/Review-Tool-User-Guide/images/1-Try-Image-2.PNG)
