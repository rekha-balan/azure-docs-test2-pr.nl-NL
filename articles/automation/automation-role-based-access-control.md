## <a name="configure-rbac-for-runbooks"></a><span data-ttu-id="2a5ae-554">Configure RBAC for Runbooks</span><span class="sxs-lookup"><span data-stu-id="2a5ae-554">Configure RBAC for Runbooks</span></span>

<span data-ttu-id="2a5ae-555">Azure Automation allows for you to assign RBAC to specific runbooks.</span><span class="sxs-lookup"><span data-stu-id="2a5ae-555">Azure Automation allows for you to assign RBAC to specific runbooks.</span></span> <span data-ttu-id="2a5ae-556">To do this run the following script to add a user to a specific runbook.</span><span class="sxs-lookup"><span data-stu-id="2a5ae-556">To do this run the following script to add a user to a specific runbook.</span></span> <span data-ttu-id="2a5ae-557">The following script can be ran by an Automation Account Admin or Tenant Admin.</span><span class="sxs-lookup"><span data-stu-id="2a5ae-557">The following script can be ran by an Automation Account Admin or Tenant Admin.</span></span>

```azurepowershell-interactive
$rgName = "<Resource Group Name>" # Resource Group name for the Automation Account
$automationAccountName ="<Automation Account Name>" # Name of the Automation Account
$rbName = "<Name of Runbook>" # Name of the runbook
$userId = "<User ObjectId>" # Azure Active Directory (AAD) user's ObjectId from the directory

# Gets the Automation Account resource
$aa = Get-AzureRmResource -ResourceGroupName $rgName -ResourceType "Microsoft.Automation/automationAccounts" -ResourceName $automationAccountName

# Get the Runbook resource
$rb = Get-AzureRmResource -ResourceGroupName $rgName -ResourceType "Microsoft.Automation/automationAccounts/runbooks" -ResourceName "$automationAccountName/$rbName"

# The Automation Job Operator role only needs to be ran once per user.
New-AzureRmRoleAssignment -ObjectId $userId -RoleDefinitionName "Automation Job Operator" -Scope $aa.ResourceId

# Adds the user to the Automation Runbook Operator role to the Runbook scope
New-AzureRmRoleAssignment -ObjectId $userId -RoleDefinitionName "Automation Runbook Operator" -Scope $rb.ResourceId
```
